About This Project

Angular Laboratory Work

This application was created as part of an Angular laboratory project to demonstrate routing and HTTP module implementation in Angular. The application features CRUD operations for albums and displays photos from each album.

Key Features

  • Routing with Angular Router
  • HTTP requests to JSONPlaceholder API
  • Observable data management
  • CRUD operations for albums
  • Responsive design with Tailwind CSS

Project Structure

The application consists of several components:

  • Home Component - Landing page
  • About Component - Project information
  • Albums Component - Album listing with delete functionality
  • Album Detail Component - Edit album information
  • Album Photos Component - View photos in an album
